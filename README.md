# 빈주머니(리팩토링 예정)
#### _개발기간 : 2023.01.16 ~. 2023.02.12
#### 총 시간 : 5576분, 약 92.9시간 소요.


"당신의 주머니는 빈 주머니 입니까? 가득 차 있습니까?  
당신의 주머니가 비어있지 않도록 지출과 수입 내역을 간편하게 기록해 보세요!"  

오늘 하루에 생겼던 지출과 수입들을 간단하게 추가하고, 내역을 간편하게 볼 수 있습니다.
